## 图标点标记

### 使用anchor定位图标
  
marker-anchor的可选值 : 
- top-left'
- top-center
- top-right
- middle-left
- center
- middle-right
- bottom-left
- bottom-center
- bottom-right

:::preview
demo-preview=./IconUseAnchor.vue
:::

### 使用offset定位图标

如果指定了anchor，将优先使用anchor。如果想使用图片内部的位置作为锚点，可使用offset。
